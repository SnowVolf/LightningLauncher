Η Lightning Launcher είναι μια γρήγορη, ελαφριά και πλήρως προσαρμόσιμη αρχική οθόνη αντικατάστασης για το Android.

Κύρια χαρακτηριστικά:
- Ευρεία ποικιλία των τρόπων διάταξης: πλέγμα, μία μόνο στήλη, οριζόντια, κάθετη, ελεύθερη τοποθέτηση, ...
- Απεριόριστη οπτική προσαρμογή: ετικέτες αλλαγής, εικόνες, γραμματοσειρές, χρώματα, ευθυγράμμιση, μεγέθη, και ούτω καθεξής.
- Καινοτόμος ιδέα: η αρχική οθόνη σας είναι σαν μια απεριόριστη επιφάνεια όπου εφαρμογές και γραφικά(widgets) μπορούν να τοποθετηθούν οπουδήποτε, με οποιοδηποτε τροπο
- Κάθε στοιχείο (εφαρμογή(app), συντόμευση(shortcut), γραφικά(widgets), ...) μπορεί να είναι ατομικά προσαρμοσμένο, κλιμάκωμένο, περιστεφόμενο, είτε για να βελτιωθεί η αναγνωσιμότητά του, ή για να αποκτήσουν κάποια πολύτιμα εικονοστοιχεία στενή διάταξη, ή απλά για διασκέδαση.
- Υποστήριξη για εξελιγμένους φακέλους, που μπορούν να περιέχουν γραφικά στοιχεία(widgets), αλλά και άλλους φακέλους με σύνθετη οργάνωση. Κάθε φάκελος μπορεί να προσαρμοστεί πλήρως, αρχίζοντας από τη διάταξη μέχρι και το οπτικό θέμα.
- Υποστήριξη για κλειδιά ρύθμισης και χειρονομίες(gestures): συνδέσετε την προτιμώμενη δράση σας με το προτιμώμενο κλειδί σας, ξεκινήστε μια προσαρμοσμένη εφαρμογή κατόπιν σύρραξης.
- Φτιάχτηκε από το μηδέν για να είναι η πιο αποτελεσματική εφαρμογή σε ενέργεια και σε πλείστη εξοικονόμηση μνήμης RAM. Όσο πιο περισσότερη μνήμη έχετε για τις εφαρμογές σας, τόσο πιο γρήγορα θα τρέξει αυτή, και η μπαταρίας σας θα έχει μεγαλύτερη διάρκεια ζωής.
- Και πολλά περισσότερα ακόμα...


Επιπλέον, η Lightning Launcher είναι δωρεάν, δεν έχει καμία διαφήμιση, δεν χρειάζεται άδεια διαδικτίου ούτε πρόσβαση διαχείρισης(root). Δεν υπάρχει πλήρης / επαγγελματική(pro) / προνομιακή ή HD έκδοση. Όλη η δύναμη της Lightning είναι αυτή εδώ και είναι διαθέσιμη σε ένα ενιαίο, πολύ μικρo και τρομερά αποτελεσματικό πακέτο. Ακόμα μπορείτε αν θελήσετε να ρίξετε μια ματιά στις επιπλέον εφαρμογές. Αγοράζοντας μια από αυτές θα έχετε επιπλέον κάποια πιο πολυτελή χαρακτηριστικά. Αυτό θα βοηθήσει επίσης στην διατήρηση αυτής τις εφαρμογής(app) στην ζωή... καθώς και του προγραμματιστή, ο οποίος είναι πρόθυμος να την κάνει το καλύτερο πρόγραμμα εκκίνησης(launcher).

Όλα τα σχόλια, εισηγήσεις και αναφορές σφαλμάτων είναι ευπρόσδεκτα. Σχόλια στο App Store είναι δύσκολο να χρησιμοποιηθούν, παρακαλώ στείλτε την ερώτησή σας / σφάλμα(bug) σε μένα στο pierrox@pierrox.net.

Άδεια:
- CALL_PHONE: απαιτούνται για συντομεύσεις για άμεση κλήση, αυτή η άδεια είναι απαραίτητη σε όλα τα προγράμματα προώθησης(launchers)που υποστηρίζουν συντομεύσεις επαφών. LL δεν θα καλέσει κανέναν για λογαριασμό σας.
- EXPAND_STATUS_BAR: όπως λέει το όνομά του, χρησιμοποιείται για την εμφάνιση της γραμμής κατάστασης(status bar).
- SET_WALLPAPER_HINTS: χρησιμοποιείται για να ρυθμίσετε την διάσταση του φόντου(wallpaper).
- Καμία άλλη άδεια δεν απαιτείται!
